Typography Roles

Three text roles cover every case. Use these instead of ad-hoc text-gray-* classes.

.type-heading — Inter 700, line-height 1.2

Page title or section header

.type-body — Poppins, line-height 1.7 (automatic on <p>)

Normal paragraph content. Readable, comfortable spacing for longer descriptions, feature copy, or instructional text.

.type-muted — Poppins, muted color, line-height 1.7
